# 项目规则

本目录是「五连杆轮腿平衡机器人的运动仿真与控制研究与实现」论文工程。

## 配图规范

- **所有插图统一使用 HTML 绘制**，输出为单文件 `.html`，位于 `figures/` 目录下，命名形如 `figN_<slug>.html`（例如 `fig1_system_framework.html`）。
- 图形元素优先使用内联 SVG；必要时辅以 CSS/Canvas，禁止依赖外部 CDN、外链字体或运行时网络资源，保证离线可渲染。
- 不要生成 PNG/JPG 位图作为正式插图；如论文导出需要位图，由用户从 HTML 自行截图，Claude 不预先生成。
- 配色与排版风格保持全篇一致：浅色背景、深色文字、坐标轴/连线使用克制的中性色，避免大面积高饱和填充。
- 每张图的 HTML 顶部用注释写明：图编号、图题、对应论文章节、关键依赖（如有）。
